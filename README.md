# Como executar o projeto?

Clone o repositório:

```bash
git clone https://github.com/llucasy/teste-front-idworks.git
```

Instale as dependências:

```bash
pnpm install
```

Inicie o servidor:

```bash
pnpm run server
```

Inicie o projeto:

```bash
pnpm run dev
```

Acesse o projeto em http://localhost:5173/
